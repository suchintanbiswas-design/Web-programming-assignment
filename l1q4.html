Submission Process

  1. Login to the course portal.
  2. Open the "Assignments" section.
  3. Upload your file.
  4. Click Submit.

File Format Rules

  • Only PDF or HTML allowed.
  • File size must be under 10MB.
  • Use proper naming conventions.

Tips

Always double-check your file before submission.
